# Next.js Project Setup

## Prerequisites
- Node.js (v14 or higher)
- npm or yarn

## Getting Started

1. **Clone the repository**
   ```sh
   git clone <repository_url>

2. **Navigate to the project directory**
   ```sh
   cd <project_directory>

3. ## Install dependencies
   ```sh
    npm install

4. ## Start the development server
    ```sh
    npm run dev
